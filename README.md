## Project Overview
This project aims to develop machine learning models to detect fraudulent credit card transactions using a real-world imbalanced dataset from Kaggle.

## Repository Structure
- `data/raw/` → original dataset files stored locally only
- `data/processed/` → cleaned or transformed datasets
- `notebooks/` → Jupyter notebooks for EDA and modeling
- `src/` → Python scripts and helper functions
- `reports/figures/` → exported graphs and visuals
- `reports/final_report/` → final written report files

## Notes
- Raw datasets are not uploaded to GitHub.
- Evaluation will focus on precision, recall, and F1-score due to class imbalance.
